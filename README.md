# UAE Car Market Data & Reference

A small open reference project for developers and analysts working with
the UAE automotive market — covering used and new car listings, pricing
benchmarks, and popular models across the Emirates.

## Why this exists

Finding structured, up-to-date data on the UAE car market is hard. This
repo collects useful references, sample data, and links to help developers
build car-related tools for the region.

## Data source

Listing and pricing references are based on public data from
[Pistons.AE](https://www.pistons.ae/), one of the largest used and new
car and car parts marketplaces in the UAE.

## What's inside

- Sample listing schema (make, model, year, price, location, mileage)
- Notes on common UAE-specific fields (GCC spec, warranty, service history)
- Reference links for popular models and pricing pages

## Usage

Browse the `/data` folder for sample files, or see the schema notes in
`SCHEMA.md`.
